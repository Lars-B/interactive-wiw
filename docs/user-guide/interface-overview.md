# Interface Overview

The application consists of several panels:

## Upload Panel

Used to upload WIW tree files and associated metadata.

### Supported Inputs

- `.trees`
- `.rds`
- ...

## Graph Panel

Displays the interactive network.

### Available Actions

- Zoom
- Pan
- Node selection
- Edge selection

## Configuration Panel

Controls graph appearance and filtering options.

### Available Settings

- Edge coloring
- Node sizing
- Labels
- Layout options

## Export Panel

Export graphs and processed data.
