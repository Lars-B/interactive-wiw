# Installation

The recommended way to use Interactive WIW Visualization is through the pre-built executables available on the GitHub Releases page.

## Download

Download the latest release for your operating system.

Available platforms:

* macOS (Intel or M)
* Linux
* Windows

Extract the downloaded archive and launch the executable.

---

## First Launch on macOS/Windows

Because the application is not code-signed, macOS may prevent it from opening the first time.

After attempting to launch the application:

1. Open **System Settings**
2. Navigate to **Privacy & Security**
3. Scroll to the Security section
4. Select **Open Anyway**

You only need to perform this step once.

On Windows you will have a similar popup.

---

## Starting the Application

After launching the executable, the application starts a local web server and opens in your default browser.

If the browser does not open automatically, navigate to:

```text
http://127.0.0.1:12712/
```

---

## Updating

To update the application:

1. Download the latest release.
2. Replace the old executable with the new version.
3. Launch the updated executable.

No additional installation steps are required.
